# Star-Core-Breakout

🚀 《星核突圍》系統重構與開發紀錄
一、 視覺與特效升級 (Visual & Effects)
雷射火控視覺重製： 修正「肚臍射擊」問題，將雷射發射原點偏移至機甲兩側槍管。實裝「圖層長度縮進」技術，讓雷射光束呈現頭尾尖銳的高能穿透視覺。

視野剝奪 (Blackout) 效能與視覺優化： 廢除舊版生硬的去背挖空，改用 HTML5 Canvas API 動態生成「邊緣模糊漸層（Soft-edged Vignette）」貼圖，並與巨型黑底容器拼接，營造深淵逼近的窒息感。

深淵蟲后 (Boss 1) 專屬武裝視覺替換：

【普攻】：替換為蟲后專屬的幽能毒液拖尾圖案（queen_base_bullet.png），並實裝彈幕轉向（Rotation）邏輯。

【骨刺彈幕】：子彈替換為閃爍藍紫光芒的巨大綠色旋轉爪子（claw_bullet.png）。

【腐蝕酸網】：替換為紫色螢光綠交織的毒液飛濺圖（acid_net.png），並會跟隨飛行軌跡旋轉。

EMP 視覺保護： 徹底移除 EMP 發動時的全螢幕閃白（Flash）效果，僅保留青色光圈擴散，防止玩家視覺疲勞。

經驗值水晶重製： 為避免與敵方彈幕（橘紅/黃色）或毒霧（綠色）混淆，將經驗球改為「菱形水晶」，並將顏色定調為高對比度的「霓虹粉紫（Magenta）」，確保在深藍星空背景下的絕對可視性。

磁力牽引特效修復： 移除了失效的外部圖片連結，改用引擎內建 Graphics 動態繪製「黃金磁力能量球」，解決綠色打叉方塊 Bug。

二、 戰鬥系統與機制擴充 (Systems & Mechanics)
動態環境災害系統 (Environmental Hazards)： 引入三種隨機災害，增加生存變數。為確保新手體驗，設定於擊殺兩隻 Boss 後（第三輪）才開始觸發。

【恆星風暴】：無視護盾的高溫輻射。經過平衡削弱，改為每 1.5 秒扣 1 滴血，伴隨機甲紅光閃爍。

【隕石碎石帶】：畫面邊緣飛出高速隕石。經過平衡削弱，限制每 800ms 生成一顆，並下修隕石體積與飛行速度至 280。

【離子風暴】：引發強烈電磁干擾，讓磁力牽引暫時失效並使特工大幅減速。

火控鎖定 AI 升級 (Boss 優先權)： 修正自動索敵（Auto-Aim）邏輯。當 Boss 進入射程時，系統會在距離計算上強制扣減權重，確保火力死死鎖定 Boss，避免被王召喚的小兵（如蟲后幼蟲）吸走子彈。

Boss 無限變大 Bug 修復： 修復了深淵蟲后在發動「虛空狂奔」時，因動畫時間差導致體積無限疊加的 Bug。改為絕對數值放大（1.8），並在衝刺結束時強制重置比例（1.5）。

Boss 技能預警機制 (Telegraphing)： 為蟲后的「虛空狂奔」加入 1.5 秒的蓄力前搖，期間 Boss 會變色、膨脹並煞車停頓，賦予玩家合理的反應與衝刺（Dash）閃避時間。

三、 數值平衡與機體調校 (Balance & Scaling)
深淵蟲后 (新手初見殺) 削弱：

基礎血量從 600 下修至 400。

普攻冷卻時間從 3000ms 延長至 4500ms，子彈飛行速度從 250 下修至 160。

大幅調降「腐蝕酸網」與「骨刺彈幕」的彈幕飛行速度，增加玩家走位容錯率。

多週目 (動態戰鬥強度係數) 實裝： 解決後期 Boss 淪為「血牛海綿怪」的問題。現在隨著輪迴數（bossCount）增加，Boss 不只血量變多，其【移動速度】、【普攻頻率】與【彈幕速度】也會按比例無上限成長，後期極具壓迫感。

幻影 (Phantom) 機體強化： 強化刺客定位，開局額外賦予【射速 +1】（Lv.2），搭配自帶 EMP 與高機動性，開場即可展現強大火力壓制。

核心特工 (Alpha) 機體重塑： 強化群控定位，開局額外賦予【射速 +1】與【分裂彈頭 +1】，第一發雷射即可產生折射，大幅改善前期清怪疲軟的劣勢。

四、 UI 與使用者體驗優化 (UI & UX)
字體剃頭 (Text Clipping) 修復： 針對「星核檔案室（圖鑑）」的中文字體頂部被裁切問題，加入 fontFamily 指定微軟正黑體，並補上 padding 防護罩。

面板排版校正： 將「異種資料庫」右側的 Boss 技能列表整體向左平移，解決文字溢出螢幕邊界的問題，達成左右視覺平衡。

暫停選單安全機制： 在暫停面板新增安全的青藍色「返回遊戲」按鈕，並將「返回主選單」按鈕向下推移且改為警告紅色，徹底杜絕玩家激戰中誤觸導致進度遺失。
